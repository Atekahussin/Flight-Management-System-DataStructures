## 📌 Project Overview
The **Flight Management System** is a comprehensive software solution designed for the airline industry to manage passenger information. It bridges the gap between passengers and managers through a menu-driven interface, handling everything from ticket purchases to complex administrative searches.

## 🏗️ Data Structure & Methodology
The core of this system is built on a **Singly Linked List** data structure.
- **Dynamic Memory Allocation:** Used to handle varying numbers of passengers efficiently without wasting memory.
- **Modular Design:** The project is divided into specialized functions for Passenger Services and Managerial Operations, ensuring code readability and maintainability.

## 🚀 Key Features
### For Passengers:
- **Ticket Purchase:** Entry of personal details, passport info, and ticket type.
- **Ticket Modification:** Capability to upgrade or downgrade ticket classes (Business, First Class, Economy) post-purchase.
- **Information Retrieval:** Printing individual passenger records.

### For Flight Managers:
- **Advanced Search:** Locate passengers instantly by **ID** or **Name**.
- **Global Records:** Displaying a comprehensive list of all passengers currently in the system.
- **Validation:** Integrated ID validation to prevent duplicate or erroneous entries.

## 🛠️ Tech Stack
- **Language:** C++
- **Concepts:** Linked Lists, Pointers, Structs, Dynamic Memory Management.
- **Environment:** Any standard C++ Compiler (e.g., MinGW, Dev-C++).
